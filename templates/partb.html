---------------Problem Statement [PART -B ] :

---->Dice Conditions

1.Die A Constraints:

Die A cannot have more than 4 spots on a face. Additionally, Die A may have multiple faces with the same number of spots.

2.Die B Constraints:

Die B can have as many spots on a face as necessary, potentially more than 6. However, the overall probability of obtaining each possible sum when rolling both dice must remain the same as before.

---->Input

->Die A:

Original configuration of Die A with spots removed. Given as a list.

->Die B:

Original configuration of Die B with spots removed. Given as a list.

---->Output :

->New Die A:

Reattached spots on Die A, respecting the constraints.

->New Die B:

Reattached spots on Die B, maintaining the same probabilities for each possible sum when rolling both dice.

---->Solution Approach

1.Calculate Current Probabilities:

Determine the probabilities of obtaining each possible sum when rolling both dice with the given configurations.

2.Adjust Die B:

Modify Die B to have enough spots on each face to maintain the same probabilities

3.Reattach Spots on Die A:

Distribute the spots on Die A to ensure no face has more than 4 spots, while maintaining fairness.

4.Output:

Return the reattached configurations for both dice as New Die A and New Die B.
