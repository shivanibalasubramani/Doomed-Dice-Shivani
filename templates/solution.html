-->"MEMOIZATION" in Probability Distribution of Dice Sums

[DYNAMIC PROGRAMMING]---->[TOP-DOWN]---->[MEMOIZATION]

Given two six-sided dice, the task is to determine the probability of each possible sum that can result from rolling both dice. This involves finding the count of possible combinations, listing all combinations, and calculating the probability of each sum.

-->Memoization Implementation in Python :

*Memoization Table:

Initially, a memoization table (memo) of size 13 is initialized. Each index of this table represents a possible sum of two dice rolls, ranging 2 to 12.

Populating Memoization Table:

The nested loops iterate over all possible combinations of dice rolls. For each combination, the sum of the dice rolls is calculated, and the corresponding entry in the memoization table is incremented to record the frequency of that sum.

Probability Calculation:

After populating the memoization table, probabilities are calculated for each possible sum. The frequency of each sum in the memoization table is divided by the total number of combinations (36) to compute the probability.
